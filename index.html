Welcome to PolyForce.

Polymer is the new project started with motive create polymer applications/elements from scratches, here we the share code and components with the world. The idea is to make learning a more open place, and later design these polymer components to work with Salesforce and bridging the gaps easily and make development look more awesome(r)

Polymer Starter Project

  • Fork this library, get some starting components and feel free to create your own branch. Submit and share you code with the world
  • Read more on : Oyecode Article to create your first polymer
    element

Fork this project and dump all folder to your native machine and run the index.html

How to run the project

  • Fork the code
  • Navigate to Project directory in Shell on Mac
  • Once in the directory, type in python -m SimpleHTTPServer to start server
  • Open Chrome or Safari and type in http://localhost:8080 to run local server
  • App should popup

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.
